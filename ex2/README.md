# (1) N次正方行列Mの要素を入力すると行列式の値を出力する

テストケースは
```python3
M = [[  3+4j, -9j , 6-5j, 8+j ],
     [ -4+5j,  7  , 2+3j, 4+j ],
     [  7-3j, 4-7j, -3j , 5-3j],
     [  1+8j, 9-6j, 1-j ,  -3 ]]
```
を使う

```bash
$ python3 ex2/python/ex2-1.py ex2/matrix.npy 
M =
[[ 3.+4.j -0.-9.j  6.-5.j  8.+1.j]
 [-4.+5.j  7.+0.j  2.+3.j  4.+1.j]
 [ 7.-3.j  4.-7.j -0.-3.j  5.-3.j]
 [ 1.+8.j  9.-6.j  1.-1.j -3.+0.j]]

det(M) = -5129.0000 -3191.0000j
```
